Interactive MNIST

Play with simple Machine Learning concepts on this site! More projects by Anish Govind can be found here. Check out how I made this site in my developer blog!

Draw a whole number (between 0 and 9) in the black square below then wait and see what number the computer thinks you have drawn! If you made a mistake you can use the Clear button to retry.

Note: This application uses machine learning, and might not be accurate 100% of the time.

The doughnut graph shows each number and the probability (determined by the computer) that the number is what you drew.

Draw a number first!
